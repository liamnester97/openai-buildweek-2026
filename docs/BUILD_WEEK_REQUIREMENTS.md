## Current Requirements

- Deadline: July 21, 2026 at 5:00 PM PDT.
- Build a working project using Codex with GPT-5.6 in one track: Apps for Your Life, Work and Productivity, Developer Tools, or Education.
- Submit a category, project description, public YouTube demo shorter than three minutes, and code repository URL.
- The demo must show the project working and include audio covering use of Codex and GPT-5.6.
- The repository must be public with relevant licensing, or private and shared with `testing@devpost.com` and `build-week-event@openai.com`.
- The README needs setup instructions, any sample data, and a clear explanation of where Codex accelerated work, key decisions, and how GPT-5.6 and Codex were used.
- The submission requires the `/feedback` Codex session ID for the session in which most core functionality was built.
