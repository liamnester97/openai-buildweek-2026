# ADR-001: One Complete Floodplain Portal

Date: 2026-07-17

## Decision

Build one complete Floodplain Memory portal with one young Stegosaurus form, four evidence outputs, museum verification, and deterministic endings.

## Rationale

The Build Week deadline requires a polished end-to-end experience. A second portal or playable Allosaurus perspective would dilute the evidence chain, accessibility work, and validation needed for the first chapter.

## Consequences

The Allosaurus perspective, multiple portals, voice, multiplayer, and procedural habitats are post-event work. Stage 1 must not add them.
